# Setup Ubuntu

### Prérequis

Ce script fonctionne sur **Ubuntu 20.04**

### Installation

Rendre le fichier executable : `sudo chmod +x install.sh`

Executer le fichier : `./install.sh`

### Logiciels installés

* Google-Chrome
* Youtube-DL
* Mpv
* SublimeText
* Install Python
* Install Pip for python
* Install GIT
* Uninstall firefox

### Settings
* Utiliser alt+right-click pour bouger et resize une fenêtre
