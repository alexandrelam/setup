# Setup Ubuntu 

### Prérequis

Ce script fonctionne sur **Ubuntu 18.04 et supérieur.**

### Installation

Rendre le fichier executable : `sudo chmod +x install.sh`

Executer le fichier : `./install.sh`

### Logiciels installés

* Google-Chrome
* Youtube-DL
* Mpv
* SublimeText
* Install Pip for python
* Uninstall python2 make python 3 default
* Install GIT
